# AGENTOS MASTER COMMAND
**Trigger Word:** AGENTOS  
**Owner:** Ingrid K Santana  
**Project:** rootedreadylife.com  

## What This System Does
Agent OS is a zero-monthly-cost automation system that manages:
- Content creation and scheduling
- Backlink outreach
- AI citation optimization

## Phases
1. GitHub repo structure (this file)
2. Notion workspace (3 databases)
3. Content generation (blog, social, email)
4. AI citation schema markup
5. Backlink outreach
6. Weekly automation commands

## Trigger Commands
| Command | What It Does |
|---------|-------------|
| AGENTOS | Run full system check and next priority |
| WEEKLYREPORT | Generate weekly status across all 3 Notion DBs |
| GENERATECONTENT | Draft all Claude-assigned content from Content Calendar |
| BUILDBACKLINKS | Draft outreach emails for Research-status targets |
| OPTIMIZECITATIONS | Test schema pages against ChatGPT/Claude/Perplexity |
