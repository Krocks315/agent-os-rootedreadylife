# Backlink Building Command
**Trigger:** BUILDBACKLINKS

## Instructions
Check the Backlink Tracker in Notion for items with Status = "Research" or "Pitched" where follow-up is due. Draft outreach emails using the templates. Update status and send or queue for Ingrid's approval.

## Steps
1. Open Notion Backlink Tracker
2. Filter: Status = "Research" (new targets to pitch)
3. For each research target:
   - Customize the appropriate outreach template
   - Save draft email
   - Update status to "Pitched"
   - Set Date Pitched to today
4. Filter: Status = "Pitched" or "Follow-up 1" where date is 7+ days old
5. For each follow-up needed:
   - Draft follow-up email
   - Update status (Pitched → Follow-up 1, Follow-up 1 → Follow-up 2)
6. Present all drafts to Ingrid for approval before sending

## Target Niches (in priority order)
1. Homesteading & gardening blogs
2. Parenting & family blogs
3. Natural wellness & herbal medicine
4. Preparedness & self-reliance
5. Sustainability & eco-living
